Frequently Asked Questions

Find answers to common questions about Flect

What is Flect and what is it used for?

Flect is a free, open-source browser extension designed specifically for designers. It provides a set of useful tools directly in your browser, helping to save time and simplify workflows.

How much does Flect cost?

Flect is completely free for all users!

Do I need to register to use Flect?

No, registration is not required. Just install the extension and start using it.

Is Flect safe for my data?

Yes, Flect is designed with user privacy in mind. We do not collect personal data without your knowledge. Since Flect's code is open-source, you can verify its safety yourself.

How to install Flect?

Find installation instructions for different browsers here.

How to activate Flect tools?

Tools are typically activated by clicking the Flect icon in the browser's extensions toolbar.

Flect isn't working, what should I do?

Ensure your browser meets the minimum requirements (see supported browsers). If the issue persists, please report it to us here.

How to use the font detector?

Activate the tool, hover over text on a page — Flect will display the font name and its characteristics.

How to use the ruler for measurements?

Activate the tool, select the mode, hold the right mouse button to measure the distance between any two points on the screen, or switch to automatic mode.

How to copy protected content?

Activate the tool, then try to select and copy the desired text.

Will new features be added?

Yes, we are actively working on Flect's development and plan to add new useful tools in future updates. Stay tuned for our news! You can also send your suggestions to our manager here.

How can I suggest a new feature or report a bug?

If you have questions or issues, you can contact our manager here.
